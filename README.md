<h2>あらゆる作品に触れた経験をシンプルに記録するツール</h2>
タイトル：ふれたもの台帳<br>
ジャンル：作品の公開メディアに限定されない履修記録メモ<br>
推奨環境：Android　Chrome　Windows<br>
進捗：アイデアを思いついた段階<br>
<br>
・読んだり見たりした作品を記録したい<br>
・映画、アニメ、小説、漫画をはじめする多種多様なメディアをメモしたい<br>
・タイトルが全部表示されるのが最重要<br>
・「映画・アニメ・小説・漫画」「まだ途中・何巻まで・完了」のカテゴリ分けができる<br>
・シンプル<br>
・テキスト優位<br>
・サムネイル表示など、自動検索系の機能は不要<br>
